## 🧠 Опис функціоналу

## ▶️ Як запустити проєкт "з нуля"
### Перед підготовка
Потрібно додати 2 файли raw_data та dataset що сюди не поміщаються 
Встановити датасет та помістити його у файл raw data

### 1. Встановлення інструментів

- Python 3.8 або новіший
- pip (менеджер пакетів Python)
- (опційно) NVIDIA GPU з CUDA для прискорення тренування

### 2. Клонування репозиторію

```bash
git clone https://github.com/your-user/powerlines-detector.git
cd powerlines-detector
```

### 3. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 4. Початкове налаштування (створення папок)

```bash
python main.py setup
```

### 5. Підготовка датасету

Помістіть анотовані зображення у форматі LabelMe JSON у папку `raw_data/`, потім:

```bash
python main.py prepare
```

### 6. Навчання моделі

```bash
# Стандартне навчання (100 епох, CPU)
python main.py train

# З власними параметрами
python main.py train --epochs 200 --batch 4 --device cuda
```

### 7. Запуск детекції

```bash
# Через графічний інтерфейс
python main.py gui

# Через командну лінію
python main.py detect photo.jpg --conf 0.25
```

## 🖱️ Інструкція для користувача (GUI)

1. **Запуск інтерфейсу** — виконати `python main.py gui`

2. **Вибір файлів**:
   - Кнопка `📂 Вибрати файли` — відкриває діалогове вікно для вибору одного або кількох зображень

3. **Налаштування детекції**:
   - Слайдер `Чутливість` — регулює поріг впевненості (від 0.1 до 0.9, за замовчуванням 0.25)

4. **Запуск обробки**:
   - Кнопка `▶ ОБРОБИТИ` — запускає детекцію у окремому потоці (інтерфейс не заморожується)

5. **Перегляд результатів**:
   - Центральне вікно відображає зображення з відміченими bounding box'ами (червоний — опора, жовтий — кабель)
   - Кнопки `← Назад` / `Вперед →` або вибір із списку для навігації між файлами

6. **Збереження**:
   - Результати автоматично зберігаються у папку `detections/`

## ⌨️ CLI — приклади команд

```bash
# Повний цикл роботи
python main.py setup                              # Ініціалізація
python main.py prepare                            # Підготовка датасету
python main.py visualize                          # Перевірка анотацій
python main.py train                              # Навчання

# Детекція
python main.py detect photo.jpg                   # Одне фото
python main.py detect images_folder/              # Папка зображень
python main.py detect video.mp4                   # Відеофайл
python main.py detect photo.jpg --conf 0.3 --model runs/detect/powerlines_detector/weights/best.pt

# Пряме використання модулів
python detect.py --source photo.jpg --conf 0.25 --iou 0.45 --device cpu
```

## 🧪 Проблеми і рішення

| Проблема | Рішення |
|----------|---------|
| `ModuleNotFoundError: ultralytics` | Виконати `pip install -r requirements.txt` |
| Низька точність на кабелях | Збільшити датасет, особливо зображення з кабелями на складному фоні |
| Повільне тренування | Встановити `--device cuda` або зменшити `--batch` до 2 |
| GUI не відкривається | Переконатися, що встановлено `tkinter` (`sudo apt-get install python3-tk`) |
| Модель не знаходиться | Перевірити шлях до `best.pt` у `runs/detect/powerlines_detector/weights/` |
